3장 데이터 구조 : 여러 데이터를 한 변수에 담는 것이 목적
 1절 리스트 : [타입과 관련없이 0개 이상의 데이터들]
  1.1 리스트 생성
	[], list(), range(from,to,step)
  1.2 기본 정보 조회
	-min(lst), max(lst), sum(lst), len(lst)
	-sorted(lst) : lst 변수 안 바뀜
	-lst.sort() : 원본이 정렬 데이터로 변경
	-lst.reverse()
  1.3 요소 추가
	- append(value), extend(lst), insert(idx,value)
  1.4 인덱싱과 요소 갯수
	lst[idx]
	lst.index[찾을데이터] : 찾을 데이터 없을시 Error
	찾을데이터 in lst
	lst.count(찾을데이터)
  1.5 원하는 요소 추출
	슬라이싱, 리스트컴프리헨션(원하는 데이터 추출)
  1.6 요소 수정하기
	인덱싱과 슬라이싱을 이용
  1.7 요소 삭제하기
	-pop(), pop(idx), remove(데이터), del lst[idx], lst.clear
  1.8 복제
	복제변수 = 원본데이터.copy()
