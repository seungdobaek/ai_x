14장 웹데이터 수집 Part1 정적
 3절 Seleninum을 이용한 동적 웹크롤링 문법
	https://selenium-python.readthedocs.io/
	로드 해야할 패키지:
		from selenium import webdriver
		from selenium.webdriver.common.by import By
	방식:
		1. Chrome 창을 연다:  driver = webdriver.Chrome()
		2. 사이트로 이동 : driver.get(url)
		3. 행동을 할 요소를 찾는다
			driver.find_element(By.찾을방법, 찾을코드)
		4. 행동 실행 (click, send_key, ...)
		5. 추출할 데이터를 찾아 가져오기
			Beautiful Soup도 사용 가능
				BeautifulSoup(driver.page_source, 'html.parser')
 4절 예제
  4.1 다음 뉴스 검색
  4.2 페이징 처리
  4.3 맞춤법검사(네이버 맞춤법 검사 이용)
