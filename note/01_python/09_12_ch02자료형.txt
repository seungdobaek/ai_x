2장 자료형과 연상자
  1절 변수
	선언 x, 할당과 동시에 타입 설정
	문자, _, 숫자 사용가능. 문자로 시작
	예약어 사용불가
	함수이름 사용하는것은 비추
	다중 변수 할당 가능(일괄 할당)
  2절 화면i/o/
	input(label) : 사용자로부터 문자열 입력
	print(출력내용들, sep='구분자', end='종결어구')
  3절 기본자료형
	숫자 : int, float, complex(복소수)
		int/int => float
		지수형표현 ex) 1.12e-20 (지수형표현은 float)
	문자 : +, * 연산
		정기표현식 : re.findall(), re.search(), re.sub()
