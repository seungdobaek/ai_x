 2절 Seaborn
	Seaborn으로 그래프 그리기 위한 단계:
		1) 데이터 준비
		2) 미적속성 설정
		3) 함수를 이용해서 그래프 그리기 (관계형=수치형, 범주형, 쌍관계, 회귀, 다중그래프)
		4) 그래프 출력(show), 저장(savefig)
  2.1 데이터 준비 : seaborn.load_dataset()을 이용하여 알려진 데이터를 가져오거나 사용할 데이터를 준비
  2.2 미적 속성 절정하기 : sns.set(미적 속성)
	그래프 스타일 지정 (style) : white, whitegrid, darkgrid, dark, ticks
	배율 (context)	         : notebook(기본값 1.0), paper(0.8), talk(1.3), poster(1.8)
	컬러팔레트 (palette) : https://matplotlib.org/tutorials/colors/colormaps.html
  2.3 seaborn 함수로 그래프 그리기
	관계형 그래프 : x축이 수치인 그래프
		scatterplot, lineplot, relplot(서브플롯제공)
	범주형 그래프 : x축이 범주형인 그래프
		stripplot(무작위로 흩뿌림), swarmplot(겹치지 않음) - 점도표
		barplot, pointplot(막대대신 marker가 있는 선) - 바 계열
		countplot(bar는 y의 평균값이 높이, countplot은 x의 빈도수 높이)
		boxplot, violinplot, boxenplot => catplot(서브플롯)
	쌍관계 그래프
		pairplot : 모든 수치형 변수 쌍에 대해 산점도와 히스토그램을 보여줌
	밀도그래프
		histplot
		displot
	회귀그래프
		regplot : hue 사용불가
		lmplot  	: hue 사용가능
