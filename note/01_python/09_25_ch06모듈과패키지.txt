6장 모듈과 패키지
 1절 파이썬 모듈 사용
	(1) import A		- A모듈 가져옴
	(2) from A import B	- A 패키지 안의 B 모듈 가져옴
	    from A,B import C	- A안의 B 모듈 안의 C 함수 가져옴
	(3) import A as B		- A모듈을 B이름으로 가져옴
	(4) from A import B as C	- A패키지안의 B 모듈을 C 이름으로 가져옴
	    from A,B import C as D	- A안의 B모듈안의 C함수를 D이름으로 가져옴
 2절 사용자 정의 모듈 : 함수나 클래스 등을 파이썬 파일에 저장하여 모듈을 정의
	모듈을 가지고오는 순서
		1. 표준모듈
		2. sys.path에 저장된 경로
			sys.path.append()를 통하여 sys.path 추가 가능
 3절 패키지
	여러 모듈을 모아둔 파일
	패키지 안에 패키지 저장 가능
	__init__.py가 꼭 있어야 한다
		__all__을 통하여 import *에서 로드할 모듈을 지정 가능
