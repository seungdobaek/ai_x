3장 분류분석
 1절. 분류분석 개요
	분류분석 : 타겟 변수가 categorical 변수인 분석
	샘플용 데이터셋 접근
		load 계열(패키지 내에 데이터 있음)
		fetch 계열(패키지 내에 데이터가 없고, 인터넷 연결시 다운)
		make 계열(가상의 데이터 생성)
 2절 분류모형의 종류
	확률적 모형 : 주어진 데이터에 각 클래스가 정답일 확률을 계산하는 모형
		확률적 생성 모형 : 각 클래스별 특정 데이터 확률분포를 추정하는 방식, 소량의 데이터로도 작동
		확률적 판별 모형 : 확률을 계산하는 함수 이용
	판별함수 모형
		주어진 데이터를 클래스에 따라다른 영역으로 나누는 경계면을 찾은 후, 
			이 경계면을 계산하는 식 => 판별함수
 3절 확률적 생성 모형
	QDA, 나이브베이지안 모형
 4절 확률적 판별 모형
	확률적 판별 모형
	의사결정나무
		여러가지 규칙을 순차저긍로 적용하면서 독립변수 공간을 분할하는 분류모형
		종속변수에 영향을 주는 독립변수들을 알아보는 목적
 5절 판별함수 모형 : 동일한 클래스가 모여있는 영역과 그 영역을 나누는 경계면을 정의
	Perceptron, SVC, 다중 퍼셉트론(인공신경망)
 6절 분류모형 성능평가
	accuracy, precision, recall, Fprate(위양성률/특이도)
	ROC curve
		위양성률(FPrate)과 재현율(TPrate/recall)의 변화를 시각화
