ch1. 머신러닝 시작하기
 1절 머신러닝 개요
	insight : 탐색적 자료분석 (EDA) : Pandas&Numpy, Matplotlib&Seaborn
	optimization : scikit-learn(머신러닝 패키지), statsmodel, scipy.stats, tensorflow, torch
	머신러닝이란? 학습을 통해 측정된 작업 성능을 향상
 2절 데이터 탐색
	변수의 종류
		명목식, 순서식, 구간식, 비율식    -  통계학의 관점
		연속형, 이산형, 범주형             - 머신러닝 관점
	통계요약 - 중심위치 측도(평균, 중앙값, 최빈값), 산포측도 (왜도, 첨도)
	데이터 EDA 가속화 (dataprep.eda 모듈이 도움이 많이 됨)
		dataprep.eda의 모듈 주요함수
			plot() : 갑 변수의 분포를 막대그래프, 파이그래프, 산점도로 표시, 요약 통계
			plot_correlation() : 변수들 사이의 상관계수를 히트맵으로 표시
			plot_missing() : 결측치 분포를 확인할 수 있돌고 표시
			plot_diff([df1, df2], x='컬럼이름') : 두 데이터프레임의 특정 컬럼의 상세 비교
	독립성 검정 (두 변수간 상호작용을 통계적으로 유의한지 검증, 두 변수가 서로 영향을 주지 않고 독립적인지)
		피서의 검정 (데이터가 적을때), 맥니마 검정
 3절 데이터 전처리
	데이터 스케일 조정, 인코딩, 원핫인코딩, 결측치 처리, 데이터 분할 (학습/테스트)
	데이터 처리 과정은 전 프로젝트의 약 60% 정도
	전체 AI 프로젝트 과정 : 데이터 수집(고객의 요구사항에 맞게) - 데이터 전처리 - 모델 구축 (ml/dl) - 
				모델 성능 평가 (A/R/P/F1/ROC) - 서비스 제작 - 배포 및 유지보수
 4절 데이터 분리
	학습용 데이터 vs 테스트용 데이터셋
		머신러닝 모형이 얼마나 신뢰도가 있는지 확인하기 위해서 학습한 이후 실제값과 예측값을 비교
