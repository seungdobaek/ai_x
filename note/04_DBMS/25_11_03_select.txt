1장 데이터베이스와 환경설정
	데이터베이스를 사용하기 위한 oracle과 sqldeveloper 설치 후 환경설정
	데이터베이스: 여러 응용 시스템들이 공휴할 수 있도록 통합, 저장된 운영데이터의 저장소
		데이터 중복을 최소화한 데이터 집합
	기능:
		정의 기능, 조작기능, 제어기능
	장점:
		데이터중복 최소화
		데이터 공유
		데이터의 무결성, 일관성 유지
		데이터의 보안 보장
	단점:
		높은 운영비
		백업과 복구
		시스템의 취약성
2장 SELECT
	저장되어있는 데이터를 가져오는 방법 (SELECT 문)
	기본 문법:
		SELECT 원하는 행 FROM 데이터 WHERE 조건
			조건이 없을경우 WHERE 생략 가능
			모든 행을 원할경우 '*' 사용
	조건에 올수 있는 연산자
		비교연산자: =, !=,^=,<>, <=, <, >=, >
		논리연산자: AND, OR, NOT
		산술연산자
		연결연산자
			산술연산자와 연결연산자의 경우 원하는행에서도 사용 가능
	SQL 연산자
		DISTINCT: 중복제거
