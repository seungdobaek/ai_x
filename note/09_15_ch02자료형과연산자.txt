2장 자료형과 연상자
 3절 기본자료형
	문자 : 문자열 인덱싱 str_[idx] / 문자열 슬라이싱 str[from:to:step]
	논리형 : True / False
		False로 간주 : [], {}, (), "", 0, None
 4절 포맷팅
	"{[인덱스]:[공백대체문자][정렬방식][자릿수][타입]}".format(변수명)
 5절 연산자
	산술연살자 : +, -, *, /, //(몫), %(나머지), **(제곱)
		/의 경우 연산결과가 항상 실수
	할당연산자 : =, +=, -=, *=, /=, //=, %=, **=
	논리연산자 : and, or, not (항상 논리연산자)
		    &, | (연산자 앞뒤에 논리값이 올경우)
	비교연산자 : <, <=, >, >=, ==, != (연산자 우선순위를 고려하여 다른 연산자와 사용)
			(&, | 보다 비교연산자가 우선순위가 낮다)
	비트연산자 : &, |, ^, ~
