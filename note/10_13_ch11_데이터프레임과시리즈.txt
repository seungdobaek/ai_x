 1절 판다스(Pandas) 패키지
	데이터 분석을 위해 필요한 패키지, 넘파이를 기반
	데이터프레임 : 2차원, 시리즈 : 1차원
	데이터 프레임 생성, 파일 io, 레이블링, 부분데이터 추출 등 다양한 장점 보유
 2절 데이터 프레임 만들기
  2.1 딕셔너리 리스트를 이용해서 데이터 프레임 만들기

  2.2 리스트를 이용해서 데이터프레임 만들기
	pd.DataFrame({ 'col1' : col1 데이터, 'col2' : col2 데이터 ...})
  2.3 read_csv (csv 파일을 데이터 프레임으로)
	pd.read_csv(csv 파일, encoding=인코딩타입, sep=구분자)
	     추가적인 arguments:
		상위 n 행만 필요시 : nrows=n
		특별행 제외 : skiprows=[제외하고 싶은 row 번호]
