ch07 스마트폰과 테블릿 레이아웃
 viewport: 웹브라우저의 HTML 문서에 특정한 정보를 나타내기 위한 태그
	viewport 속성:
		width: 화면 너비
		height: 화면 높이
		initial-scale: 초기확대 비율
		user-scalable: 확대 및 축소 가능 여부
		minimum-scale: 최소비율
		maximum-scale: 최대비율
	스마트폰과 테블릿 역시 PC화면처럼 HTML과 CSS를 만든다
		하지만 기기마다 width가 다르기에 px보다는 %, em 등의 단위를 더 자주 사용
ch08 규칙과 반응형 웹
 1절 CSS 파일 분리	
	.css파일을 따로 만드러 <link> 태그를 이용하여 html과 연결
 2절 @import 규칙
	<style> 태그 안에 @import url(.css)를 이용하여 link 태그를 대신 할수 있음
	항상 <style> 태그 내부 상단에 위치해야 함
 3절 media 규칙
	@media : 미디어 속성을 이용하여 html파일의 용도별, 디바이스별 등을 설정할 수 있음
	미디어 종류:
		all: 모든 미디어 타입
		print: 인쇄 용도
		screen: 컴퓨터 스크린 용도
		등등
 4절 반응형 웹
	@media min-device-width와 max-device-width를 이용하여 각 디바이스별 너비를 이용하여 css분리가 가능
