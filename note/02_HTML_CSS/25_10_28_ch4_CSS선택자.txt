ch4 CSS 선택자
 1절 태그선택자
	특정태그를 선택하여 CSS 속성을 적용 가능
		태그{ CSS 속성 }
 2절 전체선택자
	* 를 이용하여 모든 태그에 CSS 속성을 적용 가능
		*{ CSS 속성 }
 3절 id선택자 class 선택자
	#id{ CSS 속성 }
		html에서 id를 지정하였을경우 #를 이용하여 해당 id를 선택 가능
	.class{ CSS 속성 }
		html에서 class를 지정하였을겨우 .를 이용하여 해당 class를 선택 가능
 4절 혼합선택자
	tag, id, class를 혼합하여 CSS설정 가능
 5절 속성선택자
	선택자[속성]{ CSS 속성 } : 지정한 속성을 가지는 요소
	선택자[속성=값]{ CSS 속성 } : 지정한 속성이 해당 값을 가지는 요소
	선택자[속성^=값]{ CSS 속성 } : 속성값이 지정한 값으로 시작하는 요소
	선택자[속성$=값]{ CSS 속성 } : 속성값이 지정한 값으로 끝나는 요소
 6절 후손 및 자손 선택자
	선택자1 선택자2{ CSS 속성 } : 선택자1의 후손 중 선택자2를 가지는 요소
	선택자1>선택자2{ CSS 속성 } : 선택자1의 자식 중 선택자2를 가지는 요소
 7절 동위선택자
	선택자1+선택자2{ CSS 속성 } : 선택자1의 동위선택자(형제) 중 가장 먼저 오는 선택자2를 가지는 요소
	선택자1~선택자2{ CSS 속성 } : 선택자1의 도위선택자 중 선택자2를 가지는 요소
 8절 반응선택자
	선택자:hover{ CSS 속성 } : 마우스를 해당 선택자에 가져다 댔을경우 나타나는 속성
 9절 상태선택자
	input:focus{ CSS 속성 } : input 선택자가 focus 상태일경우
	input:disabled{ CSS 속성 } : input이 disabled 상태일경우
	iput:enabled{ CSS 속성 } : input이 enabled 상태일경우
 10절 구조선택자
	선택자:nth-child(n번째) : 선택자의 n번째
		보통 ul 또는 ol에서 사용
 11절 문자선택자, 링크선택자, 부정선택자
	잘 사용하지 않음 (기억 안날시 다른 태그들로 대체 가능)
