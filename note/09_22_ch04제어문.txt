4장 제어문
 1절 조건문 (if)
	if 조건 1:
		조건 1 True
	elif 조건 2:
		조건1 False 조건2 True
	else:
		조건 1,2 False
	elif와 else는 단독 사용 불가능

 2절 반복문
  2.1 for문
	for 변수 in 나열가능자료:
  2.2 while문
	while 조건
	
	for와 while 둘다 else를 후에 붙힐수 있으며 else는 반복문이 모두 작동하였을경우 작동
		break 시 작동 x
  2.3 break, continue
	break: 현재 반복문 탈출
	continue: 수행중인 반복문을 멈추고 조건절로 이동
