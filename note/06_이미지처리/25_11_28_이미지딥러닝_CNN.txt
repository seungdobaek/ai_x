이미지딥러닝 (CNN)
CNN - Convolutional Neural Network
	필터를 이용하여 이미지의 특징을 부각시켜 딥러닝을 시키는 방식

Tensorflow의 Conv2D Layer 객체를 이용한 Layer 만들기
	Conv2D(filter, kernel_size, padding, activation, ...)
		filter : 필터의 갯수
		kernel_size : 필터의 크기
		padding: 필터를 사용할경우 필터의 크기에 따라 output의 크기가 input보다 줄어들수 잇는데 그걸 방지
			default : valid (크기가 줄어드는걸 허용)
				same (zero-padding)
		activation : 활성화 함수
		그 외에도 strides(이동크기) 등도 정할수 있다

Pooling Layer : Conv2D를 이용할경우 Output의 수가 최대 n*n*필터수 가 되기에 크기가 너무 커져버림
	크기가 커지면 학습시 너무 오랜 시간이 걸리기에 이미지의 특징을 최대한 건드리지 않으면서 축소시키는 방식
	MaxPooling, AveragePooling 등이 있음

Flatten Layer : Conv2D와 Pooling Layer를 이용하면 3차원 데이터가 나오기에 해당 데이터를 1차원으로 변환하는 Layer
	DNN에 사용하기 전에 1차춴으로 변환할때 사용

MNIST(숫자 손글씨) 데이터를 이용하여 DNN과 CNN 비교
	비교하였을 때 CNN이 확실히 더 좋은 성능을 보여주는 걸 확인할수 있음
