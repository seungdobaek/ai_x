 4절 패키지 설치 및 삭제
	패키지 파일의 확장자는 whl

7장 객체지향프로그래밍
 1절. 객체와 클래스
	class: 속성(변수)와 메소드(함수)를 갖는 객체를 만들기 위한 설계도
	모든 class는 Object를 상속받는다
	문법: class 클래스명(상속받을클래스명):
 2절. 생성자와 소멸자
	class 내에는 생성자(생성자가 없으면 디폴트생성자 자동 생성) 및 소멸자 무조건 생성
	생성자 : __init__(self) : 객체가 생성될 때 자동 실행
	소멸자 : __del__(self) : 객체가 소멸(레퍼런스 카운트가 0이 될때)될때 자동 실행
 3절 변수와 메소드
  3.1 변수
	인스턴스 변수 
		객체별로 갖는 값을 설정할수 잇다.
	클래스 변수
		클래스 내부에 선언
		생성된 모든 객체에 동일한 값을 가진다
		객채에서 클래스 변수 수정시 해당 객체만 반영
	static 변수
		__변수명 으로 할당 / 클래스명.클래스명__static변수명 으로 참조
		클래스 내부에서만 접근하도록 하려는 용도 (모든 객체가 수정을 하더라도 동일한 값을 가진다)
  3.2 메소드
	인스턴스 메소드
	클래스 메소드 : @classmethod를 명시
	static 메소드 : @staticmethod를 명시
 4절 상속과 재정의
	상속 :  class 클래스명(상속받을클래스명:
		클래스의 재사용
		부모 클래스의 모든 속성을 사용가능
		super(): 부모클래스
