4장 RNN
	RNN : Recurrent Neural Network
	데이터의 순서가 중요하거나, 시계열 데이터일 경우
		앞의 데이터를 가지고 다음 레이어에서 훈련할때 사용
	활용분야 : 번역, 음성인식, 주가예측, 가격예측
 RNN을 이용하여 문맥상의 다음 단어 유추하기
	데이터 전처리
		문자로 되어있는 각 단어를 숫자로 변경
		keras_preprocessing.text 패키지의 Tokenizer를 이용
		각 문장별로 크기가 다르기때문에 input size가 다르다
			이를 해결하기 위해 padding을 사용
				tensorflow.keras.preprocessing.sequence 패키지의 pad_sequences 이용
	모델 학습하기
		Embedding Layer : 각 단어의 숫자를 원핫이코딩하여 임베딩후 출력
		SimpleRNN : RNN Layer
		Dense Layer
5장 LSTM
	RNN과 비슷하지만 거리가 먼 앞의 데이터의 weight을 줄이고 싶을때 사용하는 레이어
	사용방식은 RNN과 동일 SimpleRNN을 LSTM으로 변경
	예제 : LSTM을 이용한 영화 리뷰 긍정/부정 유추
