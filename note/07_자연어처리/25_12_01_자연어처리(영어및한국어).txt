1장 NLTK 자연어처리 패키지
	자연어 : 일상적인 언어
	자연어처리 : 텍스트전처리, 단어의 빈도수 측정, 문서유사도 측정, 연관분석, 딥려닝, 워드임베딩, 텍스트분류, ...
 1절 NLTK 패키지
	NLTK 패키지 : 영어 자연어 처리에 사용되는 패키지
	텍스트 전처리 : 토큰화(문장, 어절, 형태소 나누기), 정규표현식을 이용하여 불용어 처리, 어간추출
	품사태깅: 단어별 품사를 식별
 2절 형태소 분석
	형태소 : 의미가 있는 가장 작은 단위
	PosterStemmer, LanscasterStemmer, RegexpStemmer(정규표현식)
 3절 품사태깅
	pos_tag를 사용
 4절 최빈 단어 시각화
	Text 객체 이용

2장 한글 형태소 분석
 1절 자연어 처리
	자연어 이해 : 형태소 분석, 품사태깅(pos tagging), 의미분석(빈도분석, 연관분석, ...)
	자연어 이해&생성 : RNN/LSTM/GRU -> seq2seq 알고리즘 -> transformer 알고리즘
	활용분야 : 트렌드분석, 탐색적 분석, 맞춤법 검사, 번역기, 챗봇
 2절 자연어 이해/생성 절차
	1.전처리 : 단어/어절 (특수문자 제외) 
	2.분석 후보 생성 : 형태소분리, 품사태깅
	3.제약 조건 : 불용어 처리, 규칙 확인
	4.분석 : 시각화(워드클라우드, Text), 유사성 분석, 연관분석, RNN/LSTM/GUR, ...
 3절 형태소 분석 엔진
	Konlpy 패키지 (자바로 만든 패키지이기에 자바를 설치해야한다)
		HanNanum, Kkma, Komoran, Otk
	Mecab (C++로 만든 패키지) : 저사향 환경세서 사용가능
		Konlpy보다 가볍고 빠르다
	공통기능 : morphs(형태소 나누기), nouns(명사추출), pos(형태소 나누어 품사태그)
 4절 워드클라우드
	wordcloud 패키지
	wordc = WordCloud(
    		font_path=  폰트설정(한글의 경우 폰트 설정 필수),
    		background_color = 배경색,
    		relative_scaling= 단어 빈도에 따른 크기 차이 조정 (0~1사이),
    		colormap=칼라맵(matplotlib 칼라맵 사용가능),
    		max_font_size=최대 폰트 사이즈,
    		min_font_size=최소 폰트 사이즈,
		stopwords=불용어,
		mask= 워드클라우드를 씌울 이미지,
    		contour_color=테두리 색상 (mask 있을경우만 가능),
    		contour_width=테두리 두께(mask 있을경우만 가능)
	)
