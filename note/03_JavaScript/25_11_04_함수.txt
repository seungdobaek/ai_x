5장 함수
 5.1 익명함수
	var 함수이름 = function(){}
 5.2 화살표함수
	var 함수이름 = (매개변수) => {함수내용}
		매개변수가 하나일 경우 () 생략 가능
		함수 내용이 하나 일경우 {} 생략 가능
			내용이 return문 하나일 경우 return 생략 가능
 5.3 선언적 함수
	function 함수이름(){}
		웹브라우저는 script 태그 내부의 내용을 읽기 전에 선언적 함수부터 읽는다
 5.4 디폴트 값을 갖는 매개변수
	function 함수이름(매개변수1=디폴트값, 매개변수2=디폴트값, ...)
 5.5 매개변수와 리턴값
	선언된 매개변수보다 많은 매개변수로 호출시 뒷부분은 무시한다
	선언된 매개변수보다 작은 매개변수로 호출시 전달되지 않는 매개변수의 값은 undefined이다
	리턴값이 없을경우 undefined가 리턴값이다
 5.6 가변수인자함수
	arguments(매개변수의 객체)을 이용
 5.7 내부함수
	함수 내부에 함수를 지정하여 해당 함수에서만 사용할수 있도록 정의 할수 있다
 5.8 콜백함수
	매개변수에 함수를 사용할수 있다 (보통 화살표함수로 매개변수에 전달)
 5.9 타이머함수
	setTimeout(함수, 밀리초) : 밀리초 뒤에 한번 실행 (id를 리턴)
	setInterval(함수, 밀리초) : 밀리초 마다 실행 (id를 리턴)
	clearTmeout(id) : 해당 id를 가진 setTimeout을 없앤다
	clearInterval(id) : 해당 id를 가진 setInterval을 없앤다
