 6.3 toString()
	python __str__처럼 객체를 문자로 변환할때 toString()이 자동으로 실행
 6.4 속성추가 및 삭제
	추가 : 객체.속성 = 값
	삭제: delete 객체.속성

8장 기본객체
 8.1 string 내장 함수
	.length: 글자 수
	.concat() : 두 string 연결
	.substring(a,b) : a번째에서 b번째 앞까지, b생략시 끝까지
	.indexOf(), .lastindexOf()
	.trim()
	.toUpperCase(), .toLowerCase()
	.replace(), .replaceAll()
 8.2 정규표현식
	/ 정규표현 / 으로 표현가능
	정규표현식.test(str), str.match(정규표현식)
 8.3 array 내장함수
	arr1 = arr2 : 얕은복사
	arr.slice(), [...arr], Array.from(arr) : 깊은복사
	arr.sort() : ascii 순으로 오름차순
	arr.reverse() : ascii순으로 내림차순
		숫자의 경우 arr.sort((left,right) => left-right)를 이용하여 오름차순
			arr.sort((left,right) => right-left)를 이용하여 내림차순
 8.4 객체배열
	객체.sort()
		객체의 toString()의 값을 ascii 오름차순으로 정렬
		sort에 매개변수를 줌으로써 sort를 어떻게할지 설정 가능
