3장 조건문
 3.1 if문
	if(조건1){
		조건1 참일시 명령어
	} else if(조건2){
		조건1 거짓, 조건2 참일시 명령어
	} else{
		위의 조건이 전부 거짓일시 명령어
	}
 3.2 switch 문
	switch(기본자료형):
		case(기본자료형의 값1):
			기본자료형의 값이 1일경우 할 명령어
			break
		case(기본자료형의 값2):
			기본자료형의 값이 2일경우 할 명령어
			break
		default:
			위의 케이스가 아닐경우
 3.3 삼항연산자
	조건 ? 조건 참일시의 값 : 조건 거짓일시의 값
4장 반복문
 4.1 배열
	python의 list와 비슷하다
		[값1, 값2, ...]
		각 값의 type은 달라도 상관 없으며 배열 안에 배열도 가능
 4.2 for문
	for(i=처음값, 조건, 증가값)
	for(i of arr) <- arr의 index가 i로
	for(i of arr) <- arr의 값들이 i로
	arr.forEach(함수)
 4.3 while문
	do{
		반복할 명령어
	} while(조건)
		-> do가 있을경우 한번은 무조건 명령어를 수행
	while(조건){
		반복할 명령어
	}	-> 조건이 만족하면 계속 명령어 실행
