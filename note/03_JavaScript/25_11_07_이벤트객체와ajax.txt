11장 이벤트
	이벤트: 키도드를 이용해 버튼을 입력하거나 마우스 클릭과 같이 다른것에 영향을 미치는 것
	이벤트함수의 경우 this를 사용할수 있는데 파이썬의 self와 동일한 역활이다
		화살표 함수의 경우 this 사용이 불가능하다
 11.1 인라인 이벤트모델
	html에서 태그에 이벤트=이벤트함수 후 script에서 이벤트함수 정의
		이벤트는 on으로 시작
 11.2 기본이벤트모델
	script에서 태그.이벤트(이벤트함수)
		이벤트는 on으로 시작
 11.3 표준이벤트모델
	script에서 태그.addEventListener(이벤트, 이벤트함수)
		on을 뺀 이벤트
 11.4 이벤트제거
	기본이벤트모델 : 이벤트=null
	표준이벤트모델 : removeEventListener(이벤트, id)
 11.5 이벤트제한
	기본이벤트모델 : return false
	표준이벤트모델 : e.preventDefault
		e는 이벤트함수의 매개변수
