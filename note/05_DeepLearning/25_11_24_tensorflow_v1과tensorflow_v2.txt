4. XOR
	하나의 레이어(linear line)만 사용할경우 절대 cost를 60% 밑으로 낮출수 없다
	여러 layer를 이용한 Deep Learning을 이용하여 더 정확하게 훈련을 시킬수 있다
5. Deep Learning : XOR
	하나의 레이어에 input의 수*output수 만큼의 weight이 필요하며 output 수만큼의 bias가 필요
		한 레이어의 parameter 수  = input의 수*output수(weight)+output 수(bias)
	레이어 또는 각 레이어의 output의 수를 늘리면 더 높은 성공율의 학습을 볼수 있다
		하지만 이는 과적합으로 빠질수 있다

ch05 05_XOR을_tensorflow_v2
	tensorflow v2는 구글 keras 팀에서 tensorflow를 쉽게 사용할수 있도록 배포한 라이브러리
	Sequential과 같은 모델, Dense와 같은 Layer들이 정의 되어잇어서 하나하나 정의할 필요가 없다
	가능한 loss: mae(회귀분석), 
		binary_crossentropy(이진분류=로지스틱회귀분석), 
          		categorical_crossentropy(다중분류-원핫인코딩을 한 경우), 
		sparse_categorical_entropy(다중분류-원핫인코딩 안 한 경우)
